#define BUFFER_LENGTH   2048

This program implements an FTP client protocol.

The FTP client works in passive mode. This allows it to list the remote directory and lets the user copy a file to a user's hard drive. This client supports PASV, LIST, QUIT and RETR [filename] commands.
