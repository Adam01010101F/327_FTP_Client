This program implements an F\+TP client protocol. 

The F\+TP client works in passive mode. This allows it to list the remote directory and lets the user copy a file to a user\textquotesingle{}s hard drive. This client supports P\+A\+SV, L\+I\+ST, Q\+U\+IT and R\+E\+TR \mbox{[}filename\mbox{]} commands.
